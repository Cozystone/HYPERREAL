## 무엇인가

JARGEMA는 README 기준으로 웹캠 얼굴을 실시간 분석해 졸음 징후를 JDS(JARGEMA Drowsiness Score)로 표시하는 웹앱이다. 사용자가 명시적으로 동의한 경우 스냅샷을 공개 피드에 올릴 수 있다.

## 구현 범위

Next.js App Router, MediaPipe Face Mesh, EAR/PERCLOS/깜빡임/MAR/머리 자세 근사값, 반응형 대시보드, 공개 피드, 설정, 쿠키 세션, Kahoot 스타일 클래스 방 코드 생성과 라이브보드가 포함된다.

## 포트폴리오 관점

JARGEMA는 감지 기술을 장난스럽고 위험한 사회적 장면으로 옮긴 실험이다. HYPERREAL에서는 동의, 공개성, 점수화가 충돌하는 인터페이스 사례로 기록한다.
