## 무엇인가

Timevest는 README 기준으로 Android-first screen-time investing gameplay를 만드는 Expo Development Build 앱이다. 사용자의 앱 사용량을 읽어 투자/시장 게임 흐름으로 전환한다.

## 구조

React Native, Expo Development Build, TypeScript, Zustand, AsyncStorage를 사용한다. Android에서는 Kotlin UsageStats 네이티브 모듈을 준비하고, iOS는 Screen Time bridge stub과 mock fallback을 둔다.

## 포트폴리오 관점

Timevest는 시간 관리 앱을 훈계형 대시보드가 아니라 시장 게임으로 바꾸는 실험이다. 사용 시간을 수치화하되, 벌점보다 플레이 가능한 경제 시스템으로 다루는 점이 핵심이다.
