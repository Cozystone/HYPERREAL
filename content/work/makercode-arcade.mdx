## 무엇인가

MakerCode Arcade는 저장소 설명상 MakeCode 프로젝트이며 GitHub Pages로 공개되어 있다. 라이브 페이지는 최소한의 `Manual Edit`와 GitHub 연결 화면만 보여준다.

## 포트폴리오 관점

완성형 제품보다 초기 게임 제작/퍼블리시 흔적에 가깝다. HYPERREAL에서는 오래된 실험을 삭제하지 않고 아카이브 질서 안에 남기는 항목으로 둔다.
