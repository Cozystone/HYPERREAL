# HYPERREAL

Maker & Founder portfolio archive built with Next.js App Router, TypeScript, Tailwind CSS, and local MDX content.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Content

- Work entries live in `content/work/*.mdx`
- Log entries live in `content/log/*.mdx`
- Adding an MDX file creates the matching index item and detail route automatically

## Checks

```bash
npm run lint
npm run build
```
